iOS開発でちょくちょく使うコードスニペット。

スニペットというかコード例になってます。

## Usage
```
cd ~/Library/Developer/Xcode/UserData/
git clone https://github.com/sagaraya/xcode-snippets.git CodeSnippets
```

## 参考
~/Library/Developer/Xcode/UserData/CodeSnippets にある情報をもとに、Xcodeはスニペットを表示します。
